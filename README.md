# PM2.5 Forecasting using AI

## Overview
This project predicts PM2.5 pollution using deep learning (ConvLSTM + Attention).

## Features
- Spatio-temporal modeling
- Ensemble learning
- Custom loss function

## Dataset
Kaggle AISE Hackathon Dataset

## Model
ConvLSTM with residual blocks and channel attention

## Results
- Validation Loss: 0.095
- Significant improvement over baseline

## How to Run
1. Install dependencies
2. Run training script
3. Run inference
